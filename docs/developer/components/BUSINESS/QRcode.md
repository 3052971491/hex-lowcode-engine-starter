## 组件属性

| 属性       | 说明     | 类型                             | 默认值   |
| ---------- | -------- | -------------------------------- | -------- |
| content    | 内容     | string                           | ""       |
| type       | 渲染类型 | `canvas` , `svg`                 | `canvas` |
| size       | 大小     | number                           | 160      |
| bordered   | 边框     | boolean                          | true     |
| errorLevel | 纠错等级 | `L` , `M` , `Q` , `H`            | `M`      |
| status     | 状态     | `active` , `expired` , `loading` | `active` |
