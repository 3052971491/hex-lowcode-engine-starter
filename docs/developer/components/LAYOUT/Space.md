# Space 间距

设置组件之间的间距。

## 何时使用

- 适合行内元素的水平间距。
- 可以设置各种水平对齐方式。

## 组件属性

| 属性      | 说明     | 类型                                  | 默认值       |
| --------- | -------- | ------------------------------------- | ------------ |
| size      | 间距大小 | `small` ,`middle`, `large`            | `small`      |
| align     | 对齐方式 | `start` , `end` ,`center` ,`baseline` | `start`      |
| direction | 间距方向 | `vertical` ,`horizontal`              | `horizontal` |
