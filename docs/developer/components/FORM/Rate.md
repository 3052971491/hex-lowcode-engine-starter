## 组件属性

| 属性         | 说明             | 类型                                    | 默认值   |
| ------------ | ---------------- | --------------------------------------- | -------- |
| label        | 标题             | string                                  | "评分"   |
| defaultValue | 默认值           | boolean                                 | false    |
| behavior     | 表单组件显示状态 | `normal`,`disabled`,`readonly`,`hidden` | `normal` |
| tips         | 描述信息         | string                                  | -        |
| allowHalf    | 是否允许半选     | boolean                                 | false    |
| character    | 自定义字符       | string                                  | -        |
| count        | star 总数        | number                                  | 5        |
| autofocus    | 自动聚焦         | boolean                                 | false    |
